# Projeto Login e Cadastro - Angular

Projeto desenvolvido para a disciplina de Programação Web do curso de Análise e Desenvolvimento de Sistemas

## Tecnologias utilizadas
- Angular
- TypeScript
- HTML
- CSS

## Funcionalidades
- Componente de Login
- Componente de Cadastro
- Formulários básicos de autenticação

## Como executar o projeto

1. Instalar as dependências:
npm install

2. Rodar o servidor:
ng serve

3. Acessar no navegador:
http://localhost:4200
